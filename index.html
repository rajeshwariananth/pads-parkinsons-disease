PADS Smartwatch Dataset Dashboard

A browser-based dashboard for demographic, symptom, movement, sensor, tremor, wrist asymmetry, and correlation analysis. The page joins the cleaned CSV tables in pads_cleaned_csv and uses task and wrist metadata for movement comparisons.

Open with a local server, for example python3 -m http.server, then visit http://localhost:8000.
